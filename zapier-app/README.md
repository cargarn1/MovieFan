# MovieFan Zapier Integration

Zapier integration for MovieFan API.

## Setup

1. **Install dependencies:**
```bash
npm install
```

2. **Login to Zapier:**
```bash
zapier login
```

3. **Register your app:**
```bash
zapier register
```

4. **Test locally:**
```bash
zapier test
```

5. **Push to Zapier:**
```bash
zapier push
```

## Features

- **Triggers:**
  - New Room - When a new room is created
  - New Review - When a new review is posted

- **Actions:**
  - Create Room - Create a new discussion room
  - Create Review - Post a movie review

- **Searches:**
  - Search Movies - Find movies in the database

## Configuration

Make sure your MovieFan API is running on http://localhost:8001 before testing.
